一、Dev-C++ 快速上手

  1. 下载安装：DevC++ 软件下载及安装教程（详细、具体）_devc++下载-CSDN博客
  2. 新建 C 语言文件：打开 Dev-C++ → 左上角菜单栏 “文件”→“新建”→“源代码”→ 弹出编辑区。
  3. 编译运行：写完代码后，点菜单栏 “运行”→“编译运行”（或按 F11），弹出黑窗口看结果。
